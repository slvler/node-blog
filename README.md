### Node.js Blog Project



### About Create

- Url
```Json
  http://localhost:3000/about
```
- Method
```Json
  Post
```

- Request Data
```Json
{
    "title": "title",
    "description" : "description",
    "mission" : "mission",
    "vision" : "vision",
    "picture" : "picture"
}
```
